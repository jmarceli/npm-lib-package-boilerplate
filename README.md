# Usage

1. Clone this repository `git clone git@github.com:jmarceli/npm-lib-package-boilerplate.git`
2. Adjust **REAMDE.md** content (e.g. paths in badges above)
3. Remove **.git** folder
4. Add your code to **src/** folder
